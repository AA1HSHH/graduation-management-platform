# 毕业生就业检测管理系统

## 项目描述
大作业前端代码，采用vue框架，UI界面使用vuetify、element，用echarts 生成了数据统计表。 

## 引用
详见package.json

"axios": "^0.19.2",
"core-js": "^3.6.5",
"echarts": "^4.8.0",
"element-ui": "^2.13.2",
"vue": "^2.6.11",
"vue-router": "^3.2.0",
"vuetify": "^2.2.11"
